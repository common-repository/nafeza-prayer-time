Nafeza Prayer Time for shows prayer times for Muslims.

== Description ==
This Plugin displays prayer times in a table format that you can place either in a widget or in pages by shortcode.

This plugin shows prayer times with a widget. This plugin gets prayer times from the Presidency of Religious Affairs of Turkey.

= FEATURES =
* Multiple language support
* Automatically locates by client IP
* Select the calculation method
* Activate and deactivate imsak time
* Time difference control
* Browser notifications
* And others

== Installation ==
1. Upload the plugin's folder into /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Change your options from the plugin settings menu
4. All done!
